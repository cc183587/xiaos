# 外网访问部署指南

## 前置要求

1. **安装 Node.js**（如未安装，访问 https://nodejs.org/ 下载 LTS 版本）
2. **安装 Cpolar**（访问 https://www.cpolar.com/ 下载 Windows 版本）
3. **注册 Cpolar 账号**并获取 Authtoken

## 快速启动

### 一键启动（推荐）

双击运行 **`启动工厂系统-外网.bat`**

脚本会自动完成：
- ✅ 启动 Node.js 后端服务器（端口 3001）
- ✅ 启动 Cpolar 内网穿透
- ✅ 显示外网访问地址

### 手动启动

1. **启动后端服务器**
   ```bash
   cd factory-backend
   npm install  # 首次运行需要安装依赖
   npm run dev
   ```

2. **启动内网穿透**（新开一个终端窗口）
   ```bash
   cpolar http 3001
   ```

## 访问方式

### 本地访问
```
http://localhost:3001/index.html
```

### 局域网访问（同一WiFi下）
查看启动脚本输出的本机IP地址，例如：
```
http://192.168.1.100:3001/index.html
```

### 外网访问（通过 Cpolar）
在 Cpolar 窗口中找到显示的公网地址，例如：
```
https://abc123.cpolar.cn/index.html
```

将该地址发给员工即可从任何地方访问！

## Cpolar 配置（可选）

### 固定域名（需要付费账号）

如果不想每次更换地址，可以配置固定子域名：

1. 升级 Cpolar 账号（付费版支持固定域名）
2. 登录 Cpolar 后台：https://dashboard.cpolar.com/
3. 进入「隧道管理」→「预留」，创建保留域名
4. 使用以下命令启动固定域名：

```bash
cpolar http 3001 --domain=your-subdomain.cpolar.cn
```

### 配置文件方式

创建 `cpolar.yml` 配置文件：

```yaml
tunnels:
  web:
    proto: http
    addr: 3001
    subdomain: your-subdomain  # 付费版可用
    # 或使用固定域名（付费版）
    # hostname: your-subdomain.cpolar.cn
```

启动命令：
```bash
cpolar start web
```

## 注意事项

1. **Cpolar 免费版限制**：
   - 每次重启地址会变化
   - 无固定域名
   - 每日流量限制 1GB

2. **安全性**：
   - 外网访问时，建议定期修改管理员密码
   - 不要在不信任的公共网络下登录管理员账号

3. **网络要求**：
   - 你的电脑需要保持联网
   - 后端服务需要持续运行

## 故障排查

### Cpolar 无法启动
- 检查是否已安装 Cpolar
- 运行 `cpolar authtoken your-token` 登录

### 外网访问失败
- 确认 Cpolar 窗口显示 "Online" 状态
- 检查本地防火墙是否允许 3001 端口

### 后端无法启动
- 确保已运行 `npm install`
- 检查 3001 端口是否被占用：`netstat -ano | findstr :3001`

## 技术支持

如遇问题，请检查：
1. Node.js 版本：`node --version`（建议 v20+）
2. Cpolar 状态：查看 Cpolar 窗口的日志输出
3. 后端日志：查看 factory-backend 终端的输出
